# AGENTS.md

This directory contains reusable skills for product-team workflows.

Rules:
- Keep each `SKILL.md` focused on one job.
- Put deep details in `references/`.
- Put reusable templates in `assets/`.
- Keep root instructions short and operational.
- Prefer examples over abstract theory.
- When a skill becomes long, split stable policy into references and keep the main skill fast to load.
